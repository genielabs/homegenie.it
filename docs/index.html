The programmable automation system for smart devices and applications

(runs on Windows, Mac, Linux and ARM-based computers such as Raspberry Pi)

Fully-featured, yet easy

  • Setup Wizard
  • Configuration Backup/Restore
  • Customizable client UI presets
  • Package system to quickly add/remove extra features
  • Integrated smart home drivers:
      KNX, X10, ZigBee, Z-Wave
  • Scheduler with extended crontab expressions and scripting

In the video below:

  • 00:00 - 00:52 Initial setup (example with ZigBee and Z-Wave)
  • 00:53 - 02:19 Dashboard customization
  • 02:20 - 03:39 Smart thermostat with user-configurable temperature sensor and relays

Automation IDE

  • Supports C#, JavaScript, Python and visual programming with Blockly
  • Direct access to GPIO, I2C, SPI and serial ports
  • Includes .NET IoT Libraries with bindings for thousands of components
  • Fluent API for handling devices, events and data
  • Realtime data processing, transformation and analysis
  • Custom widgets using zuix.js web components

Examples included

  • Weather Widget
  • Security Alarm System
  • Energy Monitor
  • Smart Thermostat
  • Raspberry Pi: DHTxx temperature/humidity sensing and GPIO control
  • Sensor data simulator
  • Virtual modules
  • ... and more!

Quick setup

HomeGenie v1.4

Select architecture:

Enter the following commands in a terminal:

curl https://github.com/genielabs/HomeGenie/releases/download/v1.4.0-beta.24/homegenie_1.4.0-beta.24_win-x64.zip
tar -xf homegenie_1.4.0-beta.24_win-x64.zip

Enter the following commands in a terminal:

curl https://github.com/genielabs/HomeGenie/releases/download/v1.4.0-beta.24/homegenie_1.4.0-beta.24_osx-x64.zip
unzip homegenie_1.4.0-beta.24_osx-x64.zip

Enter the following commands in a terminal:

wget https://github.com/genielabs/HomeGenie/releases/download/v1.4.0-beta.24/homegenie_1.4.0-beta.24_linux-x64.zip
unzip homegenie_1.4.0-beta.24_linux-x64.zip

Enter the following commands in a terminal:

wget https://github.com/genielabs/HomeGenie/releases/download/v1.4.0-beta.24/homegenie_1.4.0-beta.24_linux-arm.zip
unzip homegenie_1.4.0-beta.24_linux-arm.zip

Enter the following commands in a terminal:

wget https://github.com/genielabs/HomeGenie/releases/download/v1.4.0-beta.24/homegenie_1.4.0-beta.24_linux-arm64.zip
unzip homegenie_1.4.0-beta.24_linux-arm64.zip

Starting the server

Set the current directory to homegenie and run the ./HomeGenie command:

cd homegenie
./HomeGenie

Open HomeGenie UI (YoT) in the web browser

     http://<server_ip>:<port>/

Where server_ip is the IP address of the machine where HomeGenie is running and port can be 80 or the first available port starting from 8080. (ex. http://192.168.1.150:8080/)
